Common Questions

  • How to create a new project?
  • How to edit an existing project?
  • How to delete a project?

How to create a new project?

To create a new project, go to the "Manage Projects" page and click the "Create New Project" button. Fill in the necessary details and submit the form.

How to edit an existing project?

To edit an existing project, click the "Edit" button next to the project you want to modify. Make the required changes and submit the updates.

How to delete a project?

To delete a project, click the "Delete" button next to the project you wish to remove. Confirm the deletion to permanently delete the project.
